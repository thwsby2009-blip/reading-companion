# Reading Companion 互動閱讀器

一個乾淨、無干擾的雙語閱讀工具。拿一段日文或英文文章，隨時點擊任何單字查意思，不用跳出頁面，不用開字典。

**線上版：** https://thwsby2009-blip.github.io/reading-companion

---

## 功能

- **點擊查單字** — 點擊文章任一單字，彈出解釋框（含讀音、詞性、中文翻譯、例句）
- **單字本** — 右手始終在文章區，左手偶爾伸向右側回顧單字
- **記住/再記** — 單字本裡可切換狀態，記住的標綠燈，再記的標黃燈
- **LocalStorage 持久化** — 單字本、閱讀進度、標記狀態，關頁面再回來還在
- **多篇文章切換** — 內建 3 篇示範文章（日文 ×2、英文 ×1）
- **響應式設計** — 手機可滑出側邊欄

---

## 設計

- 低對比度造紙術式優雅（paper-like）
- 背景暖米白（`#f5f2eb`），字深灰，少量裝飾
- 日文：Noto Serif JP / 英文：EB Garamond / UI：Inter
- 大量留白，讓閱讀區乾淨不干擾

---

## 技術

- 純 HTML + Vanilla JS，無框架依賴，單一檔案
- 內建 500+ 日文基本單字 + 基本英文單字詞庫
- LocalStorage 持久化

---

## 使用方式

1. 選一篇示範文章，或貼上自己的內容
2. 點擊任一單字查詢
3. 按「記住了」或「再記」歸類單字
4. 右側單字本隨時回顧
