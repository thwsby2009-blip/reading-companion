# 互動式語言閱讀器（Language Reading Companion）

## Concept & Vision

一個乾淨、無干擾的雙語閱讀工具。拿一段日文或英文文章，隨時點擊任何單字查意思，不用跳出頁面，不用開字典。右手始終保持在文章區，左手偶爾伸向右側回顧單字。是一個「專心閱讀，順便記單字」的工具。

## Design Language

- **Aesthetic**: 低對比度造紙術式優雅（paper-like）。背景米白，字深灰，少量裝飾。
- **Colors**:
  - Background: `#f5f2eb` (暖米白)
  - Text: `#2a2a2a`
  - Accent/Highlight: `#c4a35a` (啞光金)
  - Hover: `#e8e4d9`
  - Selected word bg: `#fff8dc`
  - Sidebar bg: `#faf8f3`
- **Typography**:
  - 日文: `'Noto Serif JP', serif`
  - 英文: `'EB Garamond', Georgia, serif`
  - UI: `'Inter', system-ui, sans-serif`
- **Spacing**: 8px base grid，大量留白讓閱讀區乾淨
- **Motion**: 最小化。彈出解釋框時 `opacity 0→1, 150ms ease-out`，無其他動畫

## Layout

```
┌─────────────────────────────────────────────────────┐
│  [文章切換 ▼]              互動閱讀器      [設定 ⚙] │
├──────────────────────────────────┬──────────────────┤
│                                  │  📝 單字本 (3)   │
│  ここはどこですか？              │                  │
│  私は東京に住んでいます。        │  ここ 這裡        │
│  今日は天気が也很好です。        │  住んで 居住      │
│                                  │  天気 天氣       │
│  Where are you from?            │                  │
│  I live in Tokyo.               │  [全部清除]      │
│                                  │                  │
└──────────────────────────────────┴──────────────────┘
```

- 左側 70%：文章閱讀區，全選文字可一鍵翻譯
- 右側 30%：單字本，滾動列表，點單字可刪除或標記
- 手機：側邊欄可滑出 overlay

## Features

### 核心功能
1. **單字點擊查詢** — 點擊文章任一單字，彈出解釋框
2. **內建詞庫** — 內含 500+ 日文基本單字 + 基本英文單字
3. **未知單字收集** — 右側單字本自動累積查過的單字
4. **標記記住/再記** — 單字本裡可切換狀態
5. **LocalStorage 持久化** — 單字本、閱讀進度、標記狀態
6. **多篇文章切換** — 內建 3 篇示範文章（日文×2、英文×1）

### 點擊解釋框內容
- 單字（本字 + 讀音 for 日文）
- 詞性
- 中文翻譯
- 簡短例句（日文附翻譯）
- 「記住了」「再記」按鈕

### 互動細節
- 點擊單字 → 0.15s 後框出現
- 點擊其他單字或文件其他處 → 框消失
- 單字本項目刪除 → 滑出動畫後移除

## Component Inventory

### 1. Header
- 左：文章下拉選單（3篇示範文）
- 中：標題「互動閱讀器」
- 右：設定按鈕（未來擴充用）

### 2. Article Panel
- 可選文字（`user-select: text`）
- `::selection` 背景色：`#c4a35a40`
- 行距 1.8，舒服的閱讀體驗
- 日文支援振り仮名（用 ruby rt 標注讀音）

### 3. Word Popup (點擊單字後彈出)
- 絕對定位，出现在點擊單字下方或上方（視位置）
- 卡片樣式，圓角 6px，陰影
- 包含：讀音、詞性、翻譯、例句、按鈕組
- 點擊外部自動消失

### 4. Vocabulary Sidebar
- 固定右側，內部滾動
- 標題：「📝 單字本 (N)」
- 每項：單字 + 翻譯 + 狀態指示燈（🟡=再記/🟢=記住）
- 刪除按鈕（×）
- 底部：清除全部按鈕

### 5. Status Bar
- 顯示：當前文章名、總單字數、記住數

## Technical Approach

- **純 HTML + Vanilla JS**，無框架依賴，單一檔案
- **詞庫格式**：JS Object `{日文/英文: {reading, pos, translation, example}}`
- **LocalStorage key**：`langreader_vocab`
- **日文分詞**：簡單正則分日本語基本單字（不含複雜形態變化）
